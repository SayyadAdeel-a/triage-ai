# TriageAI Design System 02: Ocean Breeze

**Mood:** Clear, trustworthy, calm, focused

A cool ocean-blue palette that evokes clarity and trust. Think Apple Maps on a clear day, with a fresh blue accent that feels professional yet approachable.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#F8FAFE` | Cool white with blue undertone |
| Foreground | `--color-foreground` | `#0F1729` | Deep navy ink |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#2563EB` | Vibrant ocean blue |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on blue |
| Secondary | `--color-secondary` | `#EFF4FF` | Very light blue tint |
| Muted | `--color-muted` | `#F1F5FB` | Soft sky background |
| Muted Foreground | `--color-muted-foreground` | `#64748B` | Slate gray text |
| Accent | `--color-accent` | `#2563EB` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#E0E7F1` | Light steel border |
| Sidebar | `--color-sidebar` | `#F1F5FB` | Soft sky sidebar |

## Typography

- **Display/Heading:** Inter
- **Body:** Inter
- **Mono:** JetBrains Mono
- **Heading style:** `font-semibold tracking-tight`
- **Body style:** `font-normal leading-relaxed`

## Spacing & Radius

- Base unit: 4px, section padding: `py-16 md:py-24`
- Base radius: 0.625rem (10px)

## Shadows

| Name | Use Case |
|------|----------|
| `shadow-soft` | Cards, containers |
| `shadow-elevated` | Modals, dropdowns |
| `shadow-primary` | Primary CTAs |

## Best For

- Professional and trustworthy feel
- Enterprise users who want approachable design
- Clear, focused email management
