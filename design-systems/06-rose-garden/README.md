# TriageAI Design System 06: Rose Garden

**Mood:** Elegant, warm, sophisticated, tender

A sophisticated pink-rose palette that is warm, feminine without being stereotypical, and deeply elegant. Like Apple's premium product pages with a soft floral touch.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#FFF8FA` | Cool white with rose hint |
| Foreground | `--color-foreground` | `#2D1520` | Deep plum-brown ink |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#EC4899` | Vibrant rose-pink |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on pink |
| Secondary | `--color-secondary` | `#FDF2F8` | Very light rose tint |
| Muted | `--color-muted` | `#FEF1F5` | Soft blush background |
| Muted Foreground | `--color-muted-foreground` | `#9A7080` | Muted mauve text |
| Accent | `--color-accent` | `#EC4899` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#F0DDE5` | Light rose border |
| Sidebar | `--color-sidebar` | `#FEF1F5` | Soft blush sidebar |

## Best For

- Users who appreciate elegant, refined interfaces
- Brands targeting a sophisticated, warm aesthetic
- A gentle, caring approach to email management
