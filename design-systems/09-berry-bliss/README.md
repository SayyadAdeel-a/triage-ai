# TriageAI Design System 09: Berry Bliss

**Mood:** Bold, confident, vibrant, expressive

A rich berry/magenta accent that feels bold and confident. The background stays clean while the accent commands attention. Like Apple Music's bold energy in a minimal UI.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#FBF8FF` | Cool white with berry hint |
| Foreground | `--color-foreground` | `#1F1030` | Deep purple-black ink |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#D946EF` | Vivid magenta-fuchsia |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on magenta |
| Secondary | `--color-secondary` | `#F5F0FF` | Very light violet tint |
| Muted | `--color-muted` | `#F3EDFE` | Soft lavender background |
| Muted Foreground | `--color-muted-foreground` | `#8A7498` | Muted purple-gray text |
| Accent | `--color-accent` | `#D946EF` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#E8DFF5` | Light violet border |
| Sidebar | `--color-sidebar` | `#F3EDFE` | Soft lavender sidebar |

## Key Design Decisions

- **Magenta, not purple** - a deliberate distinction from generic violet
- **Medium radius (0.75rem)** balances softness with precision
- **Purple-black foreground** adds depth without harshness

## Best For

- Bold, confident users who love expressive design
- Creative agencies and media professionals
- A design that makes a statement without screaming
