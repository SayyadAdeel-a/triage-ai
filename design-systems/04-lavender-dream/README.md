# TriageAI Design System 04: Lavender Dream

**Mood:** Creative, calm, thoughtful, graceful

A soft, real lavender palette that balances playfulness with sophistication. Not the AI-purple cliches, but a genuine lavender that feels thoughtful and graceful.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#FAF8FF` | Cool white with lavender hint |
| Foreground | `--color-foreground` | `#1E1635` | Deep plum ink |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#8B5CF6` | Thoughtful violet |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on violet |
| Secondary | `--color-secondary` | `#F0EDFF` | Very light lilac tint |
| Muted | `--color-muted` | `#F3F0FE` | Soft lavender background |
| Muted Foreground | `--color-muted-foreground` | `#7E6E9A` | Muted purple-gray text |
| Accent | `--color-accent` | `#8B5CF6` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#E4DEF5` | Light lavender border |
| Sidebar | `--color-sidebar` | `#F3F0FE` | Soft lavender sidebar |

## Best For

- Creative professionals and designers
- Users who want a calm, graceful email experience
- Teams that value thoughtfulness and beauty
