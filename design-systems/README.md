# TriageAI Design Systems Collection

10 clean, minimal, Apple-inspired design systems in vibrant light mode.

## How to Use

Replace the contents of `apps/web/src/app/globals.css` with any design system's `globals.css` file from the folder below. Each design system is a drop-in replacement that maintains full compatibility with your existing shadcn/ui components.

## Shared Design Principles

All 10 systems share these Apple-inspired foundations:

- **Light mode only** with true off-white backgrounds (never pure white)
- **One accent color** per system (except Northern Lights which has two)
- **WCAG AA compliant** contrast ratios for all text
- **Inter font family** for clean, modern typography
- **Consistent shadow system**: `shadow-soft`, `shadow-elevated`, `shadow-primary`
- **Glass effect** utility with backdrop blur for depth
- **Pill badge** base class for semantic labels
- **shadcn/ui compatible** tokens that work with existing components
