# TriageAI Design System 07: Mint Chip

**Mood:** Fresh, clean, modern, refreshing

A cool mint-teal palette that feels fresh, clean, and modern. Think Apple's Calculator app meets a spa. Cool, crisp, and instantly refreshing.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#F6FFFE` | Cool white with mint hint |
| Foreground | `--color-foreground` | `#0A2323` | Deep teal ink |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#14B8A6` | Vibrant teal |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on teal |
| Secondary | `--color-secondary` | `#ECFDF5` | Very light mint tint |
| Muted | `--color-muted` | `#F0FDFA` | Soft mint background |
| Muted Foreground | `--color-muted-foreground` | `#5E8A87` | Muted teal-gray text |
| Accent | `--color-accent` | `#14B8A6` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#D5F0EC` | Light mint border |
| Sidebar | `--color-sidebar` | `#F0FDFA` | Soft mint sidebar |

## Key Design Decisions

- **Tighter radius (0.625rem)** for a clean, crisp feel
- **Cool undertones** throughout for freshness
- **Teal as the distinguishing color** for a unique identity

## Best For

- Users who love clean, spa-like minimalism
- Health-tech and wellness-conscious users
- A refreshing break from generic email apps
