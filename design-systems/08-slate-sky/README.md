# TriageAI Design System 08: Slate & Sky

**Mood:** Professional, refined, sophisticated, clear

A sophisticated neutral palette with a vivid sky-blue accent. The restraint of the neutrals makes the blue pop. Like Apple's Pro product pages.

## Color Palette

| Role | Token | Hex | Description |
|------|-------|-----|-------------|
| Background | `--color-background` | `#FAFBFC` | True neutral off-white |
| Foreground | `--color-foreground` | `#111827` | Clean gray-black |
| Card | `--color-card` | `#FFFFFF` | Pure white surface |
| Primary | `--color-primary` | `#0EA5E9` | Vivid sky blue |
| Primary Foreground | `--color-primary-foreground` | `#FFFFFF` | White text on blue |
| Secondary | `--color-secondary` | `#F1F5F9` | Slate-100 tint |
| Muted | `--color-muted` | `#F1F5F9` | Slate gray background |
| Muted Foreground | `--color-muted-foreground` | `#64748B` | Slate-500 text |
| Accent | `--color-accent` | `#0EA5E9` | Same as primary |
| Destructive | `--color-destructive` | `#EF4444` | Clean red |
| Border | `--color-border` | `#E2E8F0` | Slate-200 border |
| Sidebar | `--color-sidebar` | `#F8FAFC` | Slate-50 sidebar |

## Key Design Decisions

- **Tightest radius (0.5rem)** for a sharp, precise feel
- **True neutrals** (no color tint) for maximum professionalism
- **Sky blue pops harder** because the surroundings are neutral

## Best For

- Enterprise users and business professionals
- Users who prefer restraint with strategic color
- A clean, serious tool that still has personality
